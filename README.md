# Course Enrollment Dashboard - Power BI Project

An interactive Power BI dashboard analyzing student enrollments,
course fees, and demographics across multiple cities in Tamil Nadu.

## 📊 Key Metrics
- **₹6.88M** Total Fees for All Courses
- **772** Total Students

## 📈 Visuals Included
- Count of Students by Course (Bar Chart)
- Count of Course Enrollments by Month
- Count of Students by City (Pie Chart)
- Count of Course by Age Group (Donut Chart)
- Sum of Fees by Course (Horizontal Bar Chart)

## 🎓 Courses Covered
MS Excel, DCA, CCA, Davinci Resolve,
Adobe After Effects, Tally ERP, Autodesk 3Ds Max, Wordpress

## 🏙️ Cities Covered
Pondicherry, Madurai, Tiruchirappalli,
Thoothukudi, Chennai, Tiruchendur

## 👥 Age Group Analysis
- 21 to 29 — 52.07% (Majority)
- 32 to 39 — 33.68%
- 41 to 44 — 9.84%
- Below 20 — Minor %
